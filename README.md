# EcoImpact 360

> La app que no solo registra residuos, sino que cambia comportamiento.

## Concepto Central

EcoImpact 360 transforma el registro de residuos escolares en un impacto ambiental tangible y medible.

Cada aula:
- Registra residuos por categoria.
- Visualiza su impacto ambiental real en equivalencias.
- Compite en ranking semanal.
- Recibe alertas si excede umbrales.
- Accede a metricas comparativas.

## El Diferenciador: Narrativa de Impacto

En lugar de solo numeros, el dashboard muestra:

> "Esta semana el colegio ha reciclado el equivalente a salvar **12 arboles**"

> "Se han evitado **8kg de CO2**"

> "El consumo de plastico bajo **18%** respecto a la semana pasada"

## Funcionalidades Clave

| Modulo | Descripcion |
|--------|-------------|
| **Registro de Residuos** | Tipo, cantidad, aula, fecha, estado |
| **Dashboard Impacto** | Totales por categoria, graficas semanales, equivalencias ecologicas |
| **Sistema de Alertas** | Umbrales dinamicos, alertas por bajo rendimiento, automaticas |
| **Ranking por Aula** | Puntuacion automatica, clasificacion en tiempo real |
| **Accesibilidad** | Alto contraste y diseno minimalista |

## Paleta de Colores (UI actual)

| Uso | Color | Hex |
|-----|-------|-----|
| Primary | Verde | `#2E7D32` |
| Secondary | Azul | `#1976D2` |
| Success | Verde claro | `#81C784` |
| Warning | Amarillo | `#FBC02D` |
| Danger | Rojo | `#D32F2F` |
| Text | Gris oscuro | `#263238` |
| Background | Gris claro | `#F5F7FA` |

**Reglas de uso**
- Primary y Success para estados positivos e impacto ambiental.
- Secondary para datos, metricas y acciones principales.
- Warning para advertencias no criticas.
- Danger exclusivamente para alertas criticas o errores.
- Text y Background como base de legibilidad y contraste.
